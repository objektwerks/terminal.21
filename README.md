Terminal21
----------
>Terminal21 scripts using Scala 3.

Server
------
>Start server before running an app! ;)
1. chmod +x server.sc ( once )
2. ./server.sc ( http://localhost:8080/ui )

Now
---
1. chmod +x now.sc ( once )
2. ./now.sc ( http://localhost:8080/ui )
